# Contributing to ResistanceMap ZA

ResistanceMap ZA is an open-source project operating under a strict zero-cost mandate. We welcome contributions from researchers, clinicians, students, and developers who share the mission of improving HIV drug-resistance surveillance in KwaZulu-Natal and the wider Southern African region.

---

## How You Can Contribute

### 1. Scientific Review

If you are a researcher with clinical or laboratory experience in HIV resistance, pharmacokinetics, or molecular epidemiology, we welcome critical review of our methodology. Please open an Issue with your feedback or contact the project lead directly.

### 2. Data Contributions

If you maintain or have access to additional public datasets relevant to KZN HIV genomic surveillance, please submit a Pull Request to docs/methodology.md noting the source and licensing.

### 3. Code Contributions

If you would like to extend the pipeline (alternative alignment algorithms, additional drug PK models, dashboard improvements), please:

- Fork the repository
- Create a new branch named feature/your-feature-name
- Submit a Pull Request with clear documentation

### 4. Documentation Improvements

Typo fixes, clarity improvements, and translation work are always welcome.

---

## Standards

- All code must be free of dependencies that require paid licences
- All written documentation must use UK English spelling and conventions
- All scientific claims must be cited or clearly marked as hypothesis
