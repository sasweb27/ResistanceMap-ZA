## Purpose of This Document

---

## Project Snapshot

ResistanceMap ZA is an open-source, zero-cost computational pipeline that maps HIV-1 drug-resistance mutation clusters across KwaZulu-Natal. It uses publicly available NCBI GenBank sequence data, cross-references mutations against the Stanford HIV Drug Resistance Database, applies pharmacokinetic modelling of first-line antiretroviral regimens, and outputs a composite Resistance Risk Index for clinical application.

The pipeline is built entirely on free infrastructure: Google Colab for computation, GitHub for hosting, and open public databases for data.

## Key Outputs Expected

- Annotated dataset of KZN HIV-1 pol gene sequences
- Mutation frequency table for top resistance variants
- Pharmacokinetic decay model for efavirenz, tenofovir, and lamivudine
- Composite RRI score table
- Public interactive dashboard

---

## Strategic Context

ResistanceMap ZA is paired with ChromaTrace KZN, a paper-based diagnostic proxy strip designed to translate the computational outputs into a physical clinic-level tool. Together they form a single integrated framework for low-cost resistance surveillance under the South African National Health Insurance rollout.
